- compression
- encoding info in fewer bits than the original info

- two main commands: gzip and bzip
  • - gzip
  • - gzip bigfile.stuff
  • - gzip -d bigfile.stuff.gz
  • - gunzip [file] can also be used to decompress files
  • - bzip2
  • - bzip2 bigfile.stuff
  • - bzip2 -d bigfile.stuff.bz2
        • - bunzip [file] can also be used to decompress files
- compressing an archive
- compressing an archive upon creation
- tar czvf home.tar.gz /[directory]
- tar cvjf home.tar.bz2 /home

- extracting an archive
- tar xzvf home.tar.gz /[directory]
- tar xvjf home.tar.bz2 /[directory]

- zip files
- can unzip windows files in linux
- yum install zip

- r: zips recursively

- zip -r [archive name] [archive]
- unzip [archive name]


- archive
- file that is a collection of a number of files/directories
- can be sorted and managed more easily than individual components
- archive backups

- tar: tape archive
- most common archive command
  • - creating an archive:
- tar cvf home.tar /home
- c: create an archive
- v: verbose output
- f: file name

- extracting an archive:
- tar xvf home.tar
- x: extract an archive
- v: verbose
- f: filename
