- su
- substitute user
- allows us to substitute another user as your own

- prompts for new user's password
- places you in the new user's shell


- .bashrc
- applies to every shell you open

- .bashprofile
- applies to login shells
- can create environment variables here
- anything run on the command line can be run from here

- user environments
- a collection of directories files and settings that set up how things look when: a user logs in
- can run commands
- can give user access to different things

- by defaul su retaubs the environment of the user who ran the command
- use: su - [user] to inherit the new user's settings

- .bash_logout
- contains commands that will be run when the user logs out

- .bash_history
- contains a list of every command made on the server

- history: prints command history
- use ctrl + r to search through your command history


Single Command
- running a single command on a substitute user
- su -c '[command]'
  • - sudo [command]
