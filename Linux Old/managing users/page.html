- manage users and groups
- new users need to be set up on a machine

- add a user account for each user
- add each user to a group
  • - add one user to a superuser group

- we will need to be logged in as root/sudo

- add the users to the server
- useradd [username]

- create superuser group
- groupadd [group name]

- set superuser group as a user's main group
- usermod -g [group name] [username]
- use id [username] to check membership

- add supplementary group for each user
- usermod - aG [group name] [username]
- use id [username] to check membership

- locking an account
- usermod -L [username]
