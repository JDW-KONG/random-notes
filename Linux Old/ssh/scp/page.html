- allows us to copy files over ssh connections
- cannot be used to run commands on the remote server

- uses and is very similar to ssh

- three uses:
- copy from remote to local
- copy from local to remote
- copy from remote to remote

- scp [file] user@server:/home/user
- transfers file to the listed directory

- scp user@server:/home/[file] [local directory]
  • - copies file from remote to specified local directory
- scp user@server:/home/user/[file] user2@server:/home/user2/
  • - copies files between remote hosts
- P: specifies port
