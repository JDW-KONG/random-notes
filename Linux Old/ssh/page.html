- secure shell

- opens a shell on another system in  a secure, encrypted manner
- replaced telnet

- l: used to specify an alternate username
- can also be done with: ssh user@hostname

- i filename: used for specifying what private key is used to authenticate this connection

- F: used for specifying what config file to use
- default: ~/.ssh/config
- this is a per user config file


SSH Private Keys
- ssh connections can be made without sending a password to be authenticaed
- this is done with key pairs

- ssh-keygen: generates ssh keys
- ssh-copy-id: copies the public key over to the target server
  • - ssh user@ip/hostname: connects to remote machine


X11 Forwarding
- allows youto tunnel gui programs over an ssh connection
  • - connection will be displayed on local machine
  • - ssh -X user@server

Other SSH Commands
- scp
- file copying utility
- replaced with scp

- sftp
- many sftp clients can connect to an ssh server
