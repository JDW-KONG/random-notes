- a file is just a  pointer to an inode that lives on a disk
- this node contains information about the data
- filename, location on disk, other metadata

- acts as a file serial number

- ls -i swap
- view inode values


Creating files
- touch: creates a zero byte file
- cp: copies file to another
- mv: moves a file from one location to another
  • - also changes file names
- rm: removes inode-filename link
- data still exists on the drive


Manipulating files
- vim, nano, ed, emacs
- saving a document creates a new inode

- finding files
- find
- specify a directory to start om
- the command will search the directory structure to find the file

- find . -name *.txt
- finds all files with names that end in '.txt'
- outputs to stdout

- common flags
- type f: find regular file
  • - type d: find directories
  • - type l: find symbolic links
  • - user: file is owned by user
