Christmas Tree Attack

- a packet has a number of flags or settings set to open
- several flags may be set to a value of 1
- a value of 1 means that a setting or flag is open

- URG, PUSH, and FIN flags are all set to 1
- means that the packet is open with high priority

- a large number of these data heavy packets can slow down or overload a network


Dynamic Link Library (DLL) Injection 
- a technique used for running code within the address space of another process
- forces the process to load a DLL
- this makes the application run in unexpected ways
- a malware DLL can be installed in another process
