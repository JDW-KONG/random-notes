Using WPS
- wifi protected setup
- originally called wifi simple config

- allows easy setup of a mobile device
- a passphrase can be complicated to a novice

- different ways to connect to a WPS AP
- the pin configured on the access point myst be entered on the mobile device
- or push a button on the access point
- near field communication: bring the mobile device close to the access point
- the usb method is no longer used

- WPS attack
- WPS is where a password is already stored and to connect to the wireless network you simply push the button
- susceptible to brute force attacks


The WPS Hack
- December 2011: WPS design flaw is discovered
- PIN is really an eight digit number
- seven digits and a checksum
- seven digits with 10,000,000 possible combinations

- the WPS process validates half of the pin
- first half, 4 digits, second half, 3 digits
  • - first half: 10,000 possiblities, seconds half 1000 possiblities
- used to take about four hours to go through all of them
- lockout and slowdown functions have now been implemented
- now takes at least 1 -7 days
