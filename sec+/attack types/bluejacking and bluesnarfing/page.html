Bluejacking and Bluesnarfing

Bluejacking
- an attacker takes control of a bluetooth device
- able to make calls and send text messages

- sending of unsolicted messages to another device via bluetooth
- NO MOBILE CARRIER IS REQUIRED

- typical functional distance is about 10 meters
- can be more or less depending on antenna and interference

- bluejack with an address book object
- instead of a contact name, write a message
- 'You are Bluejacked. Add to contacts?'

- third party software may also be used
- bloover, bluesniff


Bluesnarfing
- an attacker hijacks a bluetooth phone and extracts contact details and other sensitive information

- accessing a bluetooth enabled device and transferring data
- contact list, calendar, email, pictures, video, etc.

- first major security weakness in Bluetooth
- marcel holtmann in september 2003 and adam laurie in november 2003

- this is a serious security issue
- if you know the file, you can download it without authentication
