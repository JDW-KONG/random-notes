Zero-day Attacks
- when an exploit is found but there is no solution to prevent it
- the only way to detect a zero day exploit is when you have previously taken a baseline of your computer so that you have a standard to compare to
- allows you to check the changes since the baseline

- Zero-day
- a vulnerability that has not been detected of published
- zero-day exploits are increasingly common

- many applications have vulnerabilities
- we just haven't found them yet

- someone is working hard to find the next big vulnerability

- bad guys keep these yet to be discovered holes to themselves
