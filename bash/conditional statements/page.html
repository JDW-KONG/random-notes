passing variables to scripts at the command line
- command line arguments are referenced with: $1 - $9
- only reads as far as the first space unless quotes are used
- can also use backticks to supply an expression


the if statement
- conditionals control the flow of our code

the for loop
- repeats action x number of times

case statement
- allows us to easily test values quickly and decide how to react to them

while/do-while loops
- perform an action while a condition is true

execution operators
- && and ||
- a way of controlling how our script executes
