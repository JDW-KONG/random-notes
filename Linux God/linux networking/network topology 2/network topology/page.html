Network Topology

Switching
- switch: forwards packets within the internal network
- switch: used for lan traffic
- router: connects to internet

- the path from the client to the gateway is the default route
- the router is the gateway

- we don't need the router to transmit info on the LAN


Routing
- routers facilitate traffic between networks
- requests traverse the default route to connect to the internet
- request may traverse additional routers and networks before it reaches its destination
- use the traceroute command to view your request's route


Cloud Server
- abstracts a lot of routing into software defined networking
- a simulation of a physical network
