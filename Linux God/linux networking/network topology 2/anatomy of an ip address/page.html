Anatomy of an IP Address

- ip address: the adress of a device on it's network
- like a house number on a street

-192.168.101.100
- 32bit address
- broken into 4 8bit octets
- each octet is a 8 digit binary number
- octets can range from 0 - 255

IPv6 Example
- a lot longer than IPv4
- hexadecimal
- 128bit address
- 8 16bit sections
- addresses can be compressed by 0's


IPv4 Classful Networking
- Class A: 0.0.0.0. to 127.255.255.255
- first bit is always 0

- Class B: 128.0.0.0 to 191.255.255.255
- has a 1 in the first bit and a 0 in the second

- Class C: 192.0.0.0 to 223.255.255.255
- one for the first two bits, 0 for the third

[ Classes A - C use a portion of the 32bit address to accomodate different network sizes ]

- Class D: 224.0.0.0 to 239.255.255.255
- ones in the first three bits, 0 in the fourth
- reserved for multicast

- Class E: 240.0.0.0 to 255.255.255.255
- first four bits are all ones
- reserved for experimental use


Classes and Subnets
- Class A
- uses the remainder of the first octet to define the network, the remaining 3 are hosts
- 2,097,152 host IPs
- Class B
- uses the remainder of the first two octets to define the network, the remaining 2 are hosts
- 65,534 host IPs
- Class C
- uses the first three octets for the network and the last one for the host
- 254 host IPs


Private Networks
- Class A: 10.0.0.0 to 10.255.255.255
- Class B: 172.16.0.0 to 172.31.255.255
- Class C: 192.168.0.0 to 192.168.255.255


Reserved for Loopback
- 127.0.0.0 - 127.255.255.255

IPv6 Subnet
- 1013:18bc:0000:0000:0000:00ff:99ad:313d
- first half is network, second half is node
- first three sections are global
- fourth section is internal

- global unicast: Internet scope - routed on internet - 2001:
- unique local: Internal network scop - not routed on internet
- link local: Network link scope - not routed on internet FE80;


Network Address Translation
- a method of remapping on IP address space into another by modifying address information
- info is modified in the IP header of the packets while they are in transit across a traffic routing device

- maps external traffic to internal ip address


Conclusion
- ip addresses are necessary for Layer 3 addressing on the network
- an IPv4 address is a 32bit number separated into 4 8bit octets by periods
- an IPv6 address is a 128bit number separated into 8 16bit sections by colons
- a subnet is a subsection of the network and determines the IP range and size
- a classful network uses a portion of the IP address as the network address
- CIDR permits more flexibility in network size over the traditional classful design
