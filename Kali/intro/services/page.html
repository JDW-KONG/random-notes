Starting and Stopping a Service
- apt-get install apache2
- systemctl start apache2
- systemctl status apache2
- systemctl stop apache2
- systemctl restart apache2

Viewing Logs
- cat /var/log/[log file]
- tail -f /var/log/[log file]
- updates to file are shown in realtime


Sending a Process to the Background
- vim test1 &
- runs process in the background

- jobs
- view active jobs in back and foreground

- fg %[job id]
- brings job to foreground


List All Running Processes
- ps aux
- listing specific processes
- ps aux | grep -i apache

-monitoring processses live
- htop

Killing a Process
- kill [process id]
