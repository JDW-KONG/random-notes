Live Operating Systems

- can be booted from external media
- runs from media instead of hard drive
- no need to install to hard drive
- can also be used with virtual machines

- allows us to create a separate security domain

- Security Focused Live OS
  • - Tails
  • - Knoppix
  • - Puppy Linux
  • - JonDo
  • - TinyCore

- download the OS ISO and burn to disc or install to media
- use RUFUS, Etcher, or Pendrive Linux to create a bootable USB drive
- burn ISO to CD

- Knoppix and Tails can be run from USB and SD cards

- hit function key when PC starts to get into the BIOS
  • - search for boot options
  • - set PC to boot from USB
- create VM without virtual disk to run live OS in VM
  • - create a template for the live OS
  • - do not add virtual disk
  • - go into VM settings > storage
  • - select CD drive and mount ISO

- Live OS can be persistent or non-persistent
  • - persistent allows us to save files
    • - can be upgraded without having to download the full iso again
  • - non-persistent is more secure
    • - DVDs are by nature non-persistent
    • - no evidence is retained after the system is shut down

- USB sticks and SD cards are small and create a physical security domain
  • - use a very small usb stick for live OS
  • - can be encrypted at the hardware layer

Live Operating Systems II

- can create a Windows live CD for recovery

- Tiny Core Linux
  • - minimalist, 10MB
  • - minimizes possible attack surface
  • - works on older machines
  • - not focused on security
- Puppy Linux
  • - similar to Tiny Core
  • - noPAE is for older PCs
- Knoppix
  • - designed with security in mind
  • - debian based
  • - includes Tor proxy
  • - offers reasonable security and anonymity
  • - includes iceweasel browser
    • - rebranded firefox
  • - includes ClamAV antivirus
Jondonym
  • - designed for security
  • - debian based
  • - Tor proxy
  • - Jondonym proxy is not anonymous
  • - includes Jondofox
    • - rebranded firefox
  • - works better in Virtualbox
  • - it's similar to Tails, but with less support
  • - presents firewall options after booting into OS
  • - network connection is disabled by default
- other privacy/security focused operating systems
  • - Lightweight Portable Security
  • - Freepto
  • - Subgraph
  • - Ubuntu Privacy Mix
