Tails

- the most used privacy focused live operating system
- runs on almost any computer
- free software
- based on debian
- can use Tor or I2P
  • - defaults to Tor
- all traffic is sent through Tor

- does not store info on hard drive

- comes with many encryption options

- can be booted from VM or live media
- VM option is less secure as host and guest may contaminate each other
- works better in Virtualbox

- do not enable shared folders or clipboard

- resizable window and shared clipboard only work if VM is configured to use 32bit

- the fastest and easiest way to a reasonably secure OS that provides access to anonymizing services

- Tails + Tor can give people a false sense of security
- it is not good enough by itself

- Tails inherits the flaws of the software it comes bundled with
- no user should put full trust into any particular security solution
- trust should be distributed amongst several methods
- Tails is just a piece of an anonymizing solution

- Tails is much better than just using Tor browser in Windows
- provides isolation and a separate security domain

- Tails wont protect you from
- firmware rootkits or bios attacks
- hardware compromises
- hardware keyloggers and VGA spies
- has a unique OS fingerprint
- easily identifiable by knowledgable people

- Tails does not encrypt your documents by default
- except in the encrypted persistent volume
- provides tools to encrypt your files

- You may leave traces of your activities if you access your hard drive from tails

- Tails does not clear metadata from documents
  • - does not encrypt email by default
- Do not use the same Tails session for multiple identities
  • - connection info may correlate aliases
- Searches for Tails and downloads are logged by the NSA and other nation states
- you will be marked for more active monitoring by the government

- lack of isolation
- Torification happens within the OS
- would be safer if Torification happens on a hardware router or virtual gateway
- Tor enforcing proxy needs to be hosted on another device

- PCs running Tor have hardware information that can be linked to the user
- virtual machines hide hardware serials
- creates a virtual hard drive that only runs when the OS is running

- during startup, hit tab and enter I2P to set I2P as the anonymization method
